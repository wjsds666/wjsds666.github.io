2. 安装提醒

在Hexo根目录[Blogroot]下打开终端，输入以下指令安装必要插件

npm install --save hexo-helper-live2d
--- ## 3. 配置提醒 打开站点配置文件[Blogroot]\config.yml 搜索live2d,按照如下注释内容指示进行操作。 如果没有搜到live2d的配置项，就直接把以下内容复制到最底部。 
live2d:
  enable: true #开关插件版看板娘
  scriptFrom: local # 默认
  pluginRootPath: live2dw/ # 插件在站点上的根目录(相对路径)
  pluginJsPath: lib/ # 脚本文件相对与插件根目录路径
  pluginModelPath: assets/ # 模型文件相对与插件根目录路径
    scriptFrom: jsdelivr # jsdelivr CDN
    scriptFrom: unpkg # unpkg CDN
   scriptFrom: https://npm.elemecdn.com/live2d-widget@3.x/lib/L2Dwidget.min.js # 你的自定义 url
  tagMode: false # 标签模式, 是否仅替换 live2d tag标签而非插入到所有页面中
  debug: false # 调试, 是否在控制台输出日志
  model:
    use: live2d-widget-model-wanko # npm-module package name
    # use: wanko # 博客根目录/live2d_models/ 下的目录名
    # use: ./wives/wanko # 相对于博客根目录的路径
    # use: https://npm.elemecdn.com/live2d-widget-model-wanko@1.0.5/assets/wanko.model.json # 你的自定义 url
  display:
    position: right #控制看板娘位置
    width: 150 #控制看板娘大小
    height: 300 #控制看板娘大小
  mobile:
    show: true # 手机中是否展示

4. 生成提醒

完成后保存修改，在Hexo根目录下运行指令

【终端-2】三连 

hexo clean && hexo g && hexo s

浏览器访问 localhost:4000，右下角出现晃脑袋看板娘即成功。

5. 换模型提醒

安装新模型包 

npm install --save live2d-widget-model-xxx

仅改一行 model.use 为刚安装的包名
保存后重复【终端-2】即可

6. 卸载提醒

卸载核心插件与模型 

npm uninstall hexo-helper-live2d  
npm uninstall live2d-widget-model-包名

删除或注释整段 live2d: 配置，重复【终端-2】，页面恢复干净。

7. 尾声

看板娘不会说话，不会换装，只会晃脑袋，
却足够在量化时代给博客留一个无功能的角落。
安装完成，愿你与纸片人一起摆烂愉快。
